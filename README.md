# 세일즈맵 CRM 매핑 키 검사기

## 개요
세일즈맵 CRM에서 다른 CRM 시스템의 데이터를 이관할 때, 회사/딜/고객/리드/커스텀 오브젝트 간 매핑 키의 일치 여부를 검사하는 도구입니다.

## 주요 기능

### 1. 다중 그룹 비교 (2~10개)
- 회사 마스터, 딜, 고객, 리드, 커스텀 오브젝트 등 무제한 그룹 비교
- 엑셀/구글 시트에서 직접 복사-붙여넣기 가능

### 2. 매핑 커버리지 분석
- 각 그룹의 항목이 다른 그룹에 얼마나 매핑되는지 % 표시
- 매핑 성공/실패 항목 수 확인
- 80% 이상: 녹색, 50~80%: 노란색, 50% 미만: 빨간색

### 3. 매핑 실패 항목 경고
- 다른 그룹에서 찾을 수 없는 항목 목록 제공
- 예: "회사 마스터"에는 "삼성전자"가 있지만 "딜"에는 "(주)삼성전자"로 되어 있어 매핑 실패

### 4. 교집합 분석
- 2개 그룹 간 공통 항목
- 3개 이상 그룹에 모두 포함된 항목
- 특정 조합에만 속한 항목

### 5. 고유 항목 분석
- 각 그룹에만 존재하는 고유 항목 확인

## 설치 방법

```bash
# 의존성 설치
pip install -r requirements.txt

# 서버 실행
python app.py
```

서버가 실행되면 브라우저에서 `http://localhost:5000` 접속

## 사용 방법

1. **그룹 수 설정**: 비교할 그룹 개수를 선택 (2~10개)
2. **데이터 입력**:
   - 각 그룹의 이름을 입력 (예: "회사 마스터", "딜")
   - 엑셀/구글 시트에서 회사명 열을 복사하여 텍스트 박스에 붙여넣기
3. **분석 시작**: "분석 시작" 버튼 클릭
4. **결과 확인**:
   - 매핑 커버리지 매트릭스에서 성공률 확인
   - 매핑 실패 항목 리스트에서 문제가 있는 데이터 확인
   - 각 섹션을 클릭하여 상세 항목 확인

## 사용 예시

### 시나리오 1: 회사 마스터 ↔ 딜 매핑 검사

**회사 마스터**:
```
삼성전자
LG전자
SK하이닉스
```

**딜**:
```
삼성전자
(주)LG전자
SK하이닉스
현대자동차
```

**결과**:
- 매핑 성공: 2개 (삼성전자, SK하이닉스)
- 매핑 실패: 1개 (LG전자 ≠ (주)LG전자)
- 딜에만 있음: 현대자동차

### 시나리오 2: 3개 그룹 비교

**회사 마스터**, **딜**, **고객**을 동시 비교하여:
- 3개 모두에 있는 항목 (완벽한 매핑)
- 2개에만 있는 항목 (부분 매핑)
- 1개에만 있는 항목 (매핑 불가)

## 분석 결과 해석

### 📊 매핑 커버리지 분석
- **출발 그룹 → 대상 그룹**: 출발 그룹의 항목이 대상 그룹에 몇 % 존재하는지
- **성공률 80% 이상**: 대부분 매핑 가능 (녹색)
- **성공률 50~80%**: 일부 매핑 실패 (노란색)
- **성공률 50% 미만**: 많은 항목이 매핑 실패 (빨간색)

### ⚠️ 매핑 실패 항목
- 데이터 이관 시 연결되지 않을 항목들
- 이 항목들은 수동으로 수정하거나 매핑 규칙을 추가해야 함

### 🎯 각 그룹 고유 항목
- 해당 그룹에만 있고 다른 그룹에는 없는 항목
- 신규 데이터이거나 오래된 데이터일 수 있음

## 기술 스택
- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **분석 알고리즘**: Python Set Operations

## 주요 알고리즘

### 교집합 분석
```python
intersection = set_A & set_B  # A와 B 모두에 있는 항목
```

### 차집합 (고유 항목)
```python
unique_A = set_A - set_B - set_C  # A에만 있고 B, C에는 없는 항목
```

### 커버리지 계산
```python
coverage = len(set_A & set_B) / len(set_A) * 100  # A의 항목이 B에 몇 % 있는지
```
